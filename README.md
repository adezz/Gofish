# Gofish

自己写的，并且在实习工作期间的项目中成功用于实战钓鱼，当然可能有些部分还需要改进...  
  
基于：FLASH页面钓鱼
  
实现：  

1、感知钓鱼（自动收杆）  
2、简单的图形化操作（对于访问的目标的记录进行图形化页面删除的操作）  
3、实现数据库的存储  

缺点：  

1、没有实现同一个IP的客户端识别，自己这里没有实现，实现的方法可以通过HTTP报文头的特征来存入数据库识别，比如session的判断
